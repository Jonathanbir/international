    中華電信Inmarsat-C門號具備GMDSS，為船舶航行於國際海域及遇難救險必備功能，它可以在最短時間內自動預警與定位，GMDSS藉由衛星及通訊系統連結遇險船舶與岸上之搜救機構、遇險船舶周遭、附近航行或作業之船舶，在搜救機構接獲遇險警報號後，能在最短的時間內進行救援遇險船舶。除此之外，該系統亦能提供緊急安全通信及播送包括航行與氣象之航行安全警告等海事安全資訊。

    國際航海衛星(Inmarsat)通信業務種類繁多，包括：

    Aero、BGAN、BGAN M2M、C、FleetBroadband、GSPS、GSPS Link、IsatM2M、IsatDataPro、SwiftBroadband、Fleet Xpress、G2X等

    這些服務大致分成(第三、四代衛星(L band))及 第五、六代衛星(Ka band)新型服務，其申請程序略有不同。

    第三代衛星(L band)服務： Inmarsat C 申請

    第三代服務所使用的設備皆經過Inmarsat認證並燒錄Inmarsat機器序號(ISN)於設備內部，使用通信服務時以這組ISN進行身分辨識，因此申請門號要提供設備相關資料，才能由本分公司核配門號，將設備開通。
    詳細的申請說明請參閱 >> 申請須知

    第四代衛星(L band)服務：GSPS、BGAN、FleetBroadband

    服務的設備也有認證並核發獨一無二的IMEI碼，但使用通信時是以門號卡(SIM)內的IMSI碼來進行身分辨識。申請租用門號核可後由本分公司發給門號卡並開通門號。
    詳細的申請說明請參閱 >> 申請須知

    第五/六代衛星(Ka band)服務：Fleet Xpress / G2X

    第五/六代服務所使用的設備皆經過Inmarsat認證並燒錄Inmarsat機器序號(ISN)於設備內部，使用通信服務時以這組ISN進行身分辨識，因此申請門號要提供設備相關資料，才能由本分公司將設備開通。
    詳細的申請說明請參閱 >> 申請須知

    Inmarsat 申請須知

    申請國際航海衛星通信業務Inmarsat C 門號，請填寫中文申請書及英文申請書，附相關證件影本掛號郵寄至承辦窗口。
    請注意，非本國船籍船舶使用之門號，本公司無權限核配，須轉由國外合作業者辦理，請先洽承辦窗口確認相關細節。

    證件：

    申請人身分證明文件，公司團體請附設立登記證明文件及負責人身分證明文件，機關請於申請書蓋關防或出具公文，個人附雙證(身分證、駕照、健保卡、護照)。
    新租門號另須提供架設許可證明文件，若為船舶使用請附船籍證明文件或電台執照，非船舶使用請提供國家通訊傳播委員會之專案核准函。

    費用：

    辦理費用及匯款資訊請參閱「費率說明內的開台業務辦理費用」乙節。
    若為外國籍船舶委託代辦開台，須視該國籍開台業者報價而定，通常每部設備開台費用約為150~300美元，另外還要與該國帳務機構簽妥代理合約，帳務機構提供的帳務服務通常會收取帳戶處理年費250~500美元(每部設備)，而且需要提供保證金1000~2000美元(每艘船)。

    過戶：(更名)

    門號移轉給其他人須辦理過戶，否則帳單須由原門號持有人支付。
    如果您買船或購買二手衛星設備，賣家的門號如果尚未退租，而且門號為本分公司核配，則可以過戶方式辦理節省辦理費用。
    中文申請書請加填原用戶(異動前)相關資料，如果您辦理過戶的設備屬於GMDSS的一部分，則必須填寫英文申請書。
    為避免爭議，除了辦理開台所需證件外，請附上設備取得證明，例如船隻買賣契約、發票或原用戶切結證明。
    注意：船舶使用的門號牽涉國籍問題，請先向承辦窗口洽詢是否能夠辦理過戶。

    帳單地址變更：

    僅需要填寫中文申請書，請在申請事項勾選更址，填妥相關資料後連同相關證件影本掛號郵寄至承辦窗口。
    注意：因設備轉移需要同時變更帳單地址及用戶名稱時，須以更名(過戶)方式辦理。

    退租：

    請填寫中文申請書，申辦事項勾選退租，可指定辦理退租日期，並再核准識別碼欄填上設備門號，如果不確定有哪些門號，請洽承辦窗口代為核對。
    注意：本國籍船舶辦理退租不收費用，但需要保證清償尚未支付之通信費用。外國籍船舶退租按照原開台業者規定辦理。

    自98年1月起市話及行動撥打國際航海衛星通信僅有”870”單一接入碼

    請注意：

    自98年 1月起，由市話及行動電話撥打國際航海衛星電話僅有870可以接通，屆時871、872、873、874將全部轉為870。

    870 係不分洋區之單一接入碼，客戶僅需撥打"009＋870＋國際航海衛星電話號碼"即可自動搜尋船隻所在洋區，不需因船隻所在洋區不同而更換接入碼，提供客戶更為方便的聯絡方式。
